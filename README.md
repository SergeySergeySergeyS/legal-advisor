# 🎓 Юридический консультант для граждан

**ИИ-помощник, который отвечает на юридические вопросы простым языком со ссылками на законы и пошаговыми инструкциями.**

🔗 **Ссылка:** https://legal-advisor-XXXX.streamlit.app

## 🎯 Возможности

- 💬 **Чат-интерфейс** — задавайте вопросы как в мессенджере
- 📋 **4 категории вопросов:**
  - 🛒 Защита прав потребителей
  - 🏠 ЖКХ
  - 💼 Трудовые споры
  - 🚗 Споры с ГИБДД
- ⚖️ **Ссылки на конкретные статьи законов**
- 📝 **Пошаговые инструкции**
- 📄 **Шаблоны документов** (скоро)

## 💰 Тарифы

| Тариф | Возможности | Цена |
|-------|-------------|------|
| 🆓 **Free** | 3 вопроса в день | Бесплатно |
| 💎 **Premium** | Безлимитные вопросы + все категории | 299 ₽/мес (скоро) |

## 🚀 Как пользоваться

1. Откройте ссылку в браузере
2. Выберите категорию вопроса
3. Опишите вашу ситуацию
4. Получите консультацию со ссылками на законы
5. Следуйте пошаговой инструкции

## 🛠️ Технологии

- 🐍 Python + Streamlit
- 🤖 GigaChat (Сбер) — ИИ-модель
- ☁️ Streamlit Cloud — хостинг
- 🔐 Streamlit Secrets — безопасное хранение ключей

## ⚖️ Важно

Консультации носят информационный характер и не заменяют очную консультацию юриста.
Для сложных случаев рекомендуется обратиться к профессиональному юристу.
